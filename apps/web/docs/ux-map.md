## Key Interactions

1. **Drag & Drop**: Kanban cards, time blocks, sidebar reordering
2. **Inline Editing**: Task titles, descriptions
3. **Quick Actions**: Checkbox toggle, status change from list
4. **Keyboard Shortcuts**: 
   - `n`: New task
   - `t`: Go to Today
   - `c`: Go to Calendar
   - `/`: Focus search
   - `Esc`: Close modal
5. **@Mentions**: Autocomplete in comments
6. **Real-time Updates**: WebSocket for live collaboration
